# Vistapool, Sugarvalley and Hayward

Add support for Vistapool devices such as Hayward Aquarite, Sugar Valley, Hidrolife, DA-GEN, Vistapool, Poolwatch and Kripsol. Monitor water temperature, pH, ORP, hydrolysis and salt cell status in real time. Control filtration, lighting, backwash, pool cover, auxiliary relays and chlorination shock directly from Homey. Automate your pool with flow cards for all actions, triggers and conditions.
